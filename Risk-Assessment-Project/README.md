# 📊 Risk Assessment Project

## 🏢 Scenario
A mid-sized fintech company that stores customer financial and personal data.

## 🎯 Objective
Identify, assess, and reduce cybersecurity risks using a structured risk assessment methodology.

## 🧱 Methodology
- Asset Identification  
- Threat Identification  
- Vulnerability Analysis  
- Risk Scoring  
- Mitigation Planning  

## 📊 Sample Risks Identified
- Data breach due to weak encryption  
- Phishing attacks targeting employees  
- Web application vulnerabilities  

## 🛡️ Mitigation Strategies
- Implement strong encryption controls  
- Conduct security awareness training  
- Secure web applications against common attacks  

## 📌 Framework Alignment
Based on NIST Cybersecurity Framework (CSF)

## ✅ Outcome
Developed a structured risk register and mitigation plan to reduce organizational risk exposure.
